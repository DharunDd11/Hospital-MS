# 🏥 Hospital Management System (HMS)

for managing hospital operations efficiently, built using:

- ⚙️ **Backend:** Java 
- 🗄️ **Database:** MySQL

 ## 🚀 Features

### 🧑‍⚕️ Admin / Staff Can:
- Manage doctors, patients, appointments
- View patient details and reports


### 👩‍⚕️ Doctors Can:
- View their schedules
- Access assigned patients
